# Week 05 - Exercise 02: Docker Compose Practice

This exercise is organized into 3 parts:
- Part 1: Basic Docker Compose commands
- Part 2: 15 compose file practices
- Part 3: 10 advanced compose exercises

## Quick Validate

Use these commands in each exercise folder:

```bash
docker compose config
docker compose up -d
docker compose ps
docker compose down -v
```

## Folder Layout

- `part1`: command cheat sheet
- `part2`: `bai01` to `bai15`
- `part3`: `baitap01` to `baitap10`
